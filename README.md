\# 🏢 Sistema de Atención a Personas - Nutrisco



Sistema inteligente de gestión RRHH con IA para automatizar atención a colaboradores y análisis de métricas.



!\[Python](https://img.shields.io/badge/Python-3.11-blue)

!\[Streamlit](https://img.shields.io/badge/Streamlit-1.39-red)

!\[License](https://img.shields.io/badge/License-Private-yellow)



\## 🎯 Características Principales



\### Para Empleados (Front-end)

\- 🤖 \*\*Chatbot 24/7\*\* con IA (GPT-4) que responde consultas sobre:

&nbsp; - Licencias y permisos

&nbsp; - Beneficios y liquidaciones

&nbsp; - Contratos y documentos

&nbsp; - Políticas internas

&nbsp; - Sistema BUK



\### Para Equipo RRHH (Back-end)

\- 📊 \*\*Dashboard Ejecutivo\*\* con KPIs en tiempo real

\- 🔍 \*\*Búsqueda Avanzada\*\* de historial de conversaciones

\- 📑 \*\*Análisis IA\*\* para detectar temas emergentes

\- 📈 \*\*Métricas Automáticas\*\* de eficiencia
